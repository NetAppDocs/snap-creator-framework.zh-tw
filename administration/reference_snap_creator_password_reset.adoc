= SnapCreator密碼重設
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您忘記Snap Creator密碼、可以重設密碼。

若要重設您的Snap Creator密碼、

. 瀏覽至scServer/enger/etc資料夾。
. 編輯snapcreator.properties檔案。
. 在SNAPCREATOR_pass參數中輸入新密碼。
+

NOTE: 您可以用純文字提供密碼。密碼會自動加密。

. 按一下「 * 儲存 * 」。
