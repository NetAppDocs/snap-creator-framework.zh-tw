= 安裝Snap Creator代理程式
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在Windows和UNIX主機上安裝SnapCreator Agent。


IMPORTANT: 雖然允許在單一主機上安裝及執行多個代理程式、但最佳做法是每個主機使用單一代理程式。

如果您打算使用IBM Domino外掛程式、則外掛程式需要將Snap Creator Agent安裝在Domino資料夾以外的位置。
